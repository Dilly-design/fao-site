# FAO — Landing Page (fao.sa)

Static landing page for FAO Catering Company (شركة فاو الذواقة لتقديم الوجبات), Riyadh.

- **Stack:** single static `index.html` — no build step.
- **Brand system:** deep forest green `#1B5148`, commercial orange `#EB5B00`, warm gold `#FFB200`, cream `#FFE6A9`; Anaqa (Arabic) + Nunito (English/numerals).
- **Photography:** Kroma Studio, Feb 2025 shoot (web-optimized in `assets/img/`).
- **WhatsApp number:** single constant `WA_NUMBER` at the top of the `<script>` block in `index.html`.

## Deploy
Hosted on GitHub Pages. Any push to `main` redeploys within ~2 minutes.

```bash
git add -A && git commit -m "update" && git push
```
